How to run this program:

A) To run on linux distributions
	1) Change to the directory that contains all the source files, txt files(instructions.txt & source programs)
	2) Enter: make -f makefile.mak
	3) Enter: ./assembler "one operand instructions file.txt here" "two operand instructions file.txt here"
	4) Press enter
	5) The program should prompt for the source assembly files, so enter a valid .txt file that contains assembly code
	
	**NOTE**: This program is designed to run on assembly files that contain "START" as the first line
	in the assembly code.
